# 自然语言处理 Natural Language Processing

本页用于整理 NLP 方向的重要论文、综述、教程与代码资源。

## 入门综述 / 学习入口

### Stanford CS224N
- 链接：<https://web.stanford.edu/class/cs224n/>
- 简介：NLP with Deep Learning 经典课程。
- 推荐理由：适合从词向量、RNN 一路过渡到 Attention、Transformer 和预训练模型。

## 经典论文

### Attention Is All You Need
- 链接：<https://arxiv.org/abs/1706.03762>
- 简介：提出 Transformer 架构。
- 推荐理由：NLP、视觉、多模态、LLM 的共同祖先论文之一。

### BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding
- 链接：<https://arxiv.org/abs/1810.04805>
- 简介：经典预训练语言模型论文。
- 推荐理由：理解 encoder-only 预训练范式的重要起点。
