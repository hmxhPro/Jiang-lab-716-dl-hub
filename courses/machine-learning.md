# 机器学习课程

## 入门课程

### 吴恩达 Machine Learning Specialization
- 链接：<https://www.coursera.org/specializations/machine-learning-introduction/?msockid=1a679f32dc2469372db8897edddb685c>
- 简介：机器学习入门经典课程。
- 推荐理由：适合打基础，帮助建立监督学习、无监督学习和模型调参的直觉。
- 适合人群：机器学习初学者。

### Stanford CS229（可免费学习版本）
- 链接：<https://see.stanford.edu/Course/CS229>
- 课程主页：<https://cs229.stanford.edu/?C=M%3BO%3DA>
- 简介：偏理论与推导的经典机器学习课程。
- 推荐理由：适合认真补数学推导和算法原理。
- 适合人群：希望把 ML 基础打扎实的同学。

### 李宏毅 Machine Learning 2021
- 官方课程页：<https://speech.ee.ntu.edu.tw/~hylee/ml/2021-spring.php>
- B站镜像：<https://www.bilibili.com/video/BV1rm4y1R7vY/>
- 简介：中文机器学习课程里非常受欢迎的一套。
- 推荐理由：讲解深入浅出，覆盖 Transformer、自监督学习、RL 等内容。
- 适合人群：希望用中文系统学习 ML / DL 的同学。

## 配套讲义 / 笔记

### 《机器学习》周志华（西瓜书）
- 链接：<https://www.tup.tsinghua.edu.cn/bookscenter/book_06402703.html>
- 简介：中文机器学习经典教材。
- 推荐理由：体系完整，适合建立中文语境下的 ML 知识框架。
- 适合人群：本科生 / 研究生入门。

### 《统计学习方法》李航
- 链接：<https://www.tup.tsinghua.edu.cn/Wap/tsxqy.aspx?id=08132901>
- 简介：偏统计学习方法与算法推导的经典教材。
- 推荐理由：尤其适合补 SVM、EM、HMM、CRF、提升方法等传统基础。
- 适合人群：想补理论的同学。

### Pattern Recognition and Machine Learning（PRML）
- 链接：<https://www.microsoft.com/en-us/research/publication/pattern-recognition-machine-learning/?msockid=22c7c8ba155566462595decd143b6791>
- 简介：Christopher Bishop 的经典教材。
- 推荐理由：适合理解概率图模型、贝叶斯方法和统计建模思想。
- 适合人群：进阶学习者。
