# 优秀 GitHub 仓库

## 教学类仓库

### D2L 中文版
- 链接：<https://github.com/d2l-ai/d2l-zh>
- 简介：《动手学深度学习》中文仓库。
- 推荐理由：书、代码、讨论一体化，适合长期跟学。

### OpenAI Spinning Up
- 链接：<https://github.com/openai/spinningup>
- 官方介绍：<https://openai.com/index/spinning-up-in-deep-rl/>
- 简介：OpenAI 的强化学习教学资源。
- 推荐理由：适合系统学习 PPO、SAC、DDPG、TD3 等 RL 基础。

## 论文复现仓库

### Hugging Face Transformers
- 链接：<https://github.com/huggingface/transformers>
- 简介：NLP、视觉、音频和多模态模型最常用的开源库之一。
- 推荐理由：学习 Transformer、BERT、ViT、LLM 很实用。

### MMDetection
- 链接：<https://github.com/open-mmlab/mmdetection>
- 简介：经典目标检测与实例分割工具箱。
- 推荐理由：做 CV 实验时非常常用，适合快速搭 baseline。

### Turoad LaneDet
- 链接：<https://github.com/Turoad/lanedet>
- 简介：车道线检测工具箱，集成了多种 lane detection 方法。
- 推荐理由：如果组里做车道线检测，这个仓库很值得长期维护关注。

## 工程模板 / 研究框架仓库

### Stable-Baselines3
- 链接：<https://github.com/DLR-RM/stable-baselines3>
- 简介：PyTorch 版强化学习算法库。
- 推荐理由：文档成熟、社区活跃，做 RL baseline 非常方便。

### CleanRL
- 链接：<https://github.com/vwxyzjn/cleanrl>
- 简介：单文件、高可读性的 RL 实现集合。
- 推荐理由：适合读源码、快速改算法。

### CityFlow
- 链接：<https://github.com/cityflow-project/CityFlow>
- 论文：<https://arxiv.org/abs/1905.05217>
- 简介：大规模城市交通信号控制仿真环境。
- 推荐理由：做交通信号灯控制方向时非常关键。
