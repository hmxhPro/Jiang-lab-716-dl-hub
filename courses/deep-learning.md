# 深度学习课程

## 中文 / 双语优质课程

### 李沐《动手学深度学习》
- 链接：<https://d2l.ai/>
- B站视频：<https://www.bilibili.com/video/BV18h411r7Z7/>
- 配套代码：<https://github.com/d2l-ai/d2l-zh>
- 简介：经典的“书 + 代码 + 视频”一体化深度学习入门资源。
- 推荐理由：非常适合中文学习者，强调边学边写代码，覆盖 CNN、RNN、Attention、CV、NLP 等核心内容。
- 适合人群：初学者 / 希望系统补齐 PyTorch 实战能力的同学。

### DeepLearning.AI Deep Learning Specialization（Andrew Ng）
- 链接：<https://www.coursera.org/specializations/machine-learning-introduction/?msockid=1a679f32dc2469372db8897edddb685c>
- 简介：吴恩达团队的经典深度学习专项课程。
- 推荐理由：讲解清晰，适合建立神经网络、优化、CNN、序列模型的整体框架。
- 适合人群：完成机器学习基础后，准备正式进入深度学习的同学。

## 经典公开课

### Stanford CS231n: Deep Learning for Computer Vision
- 链接：<https://cs231n.stanford.edu/2023>
- 简介：深度学习与计算机视觉领域最经典的课程之一。
- 推荐理由：课件和作业都很扎实，适合建立 CV 方向的完整知识体系。
- 适合人群：想系统学习 CNN、检测、分割、视觉表征学习的同学。

### fast.ai — Practical Deep Learning for Coders
- 链接：<https://course20.fast.ai/>
- 简介：偏工程实践的深度学习课程。
- 推荐理由：上手快，强调把模型真正跑起来，适合想尽快做项目的人。
- 适合人群：有一定 Python 基础，想快速进入实战的同学。

## 配套讲义 / 书籍

### Dive into Deep Learning（D2L.ai）
- 链接：<https://d2l.ai/>
- 中文仓库：<https://github.com/d2l-ai/d2l-zh>
- 简介：可在线阅读、可运行代码的开源深度学习教材。
- 推荐理由：把数学、代码和实验统一在一个资源里，学习效率很高。
- 适合人群：入门到进阶。

### 《Deep Learning》— Ian Goodfellow / Yoshua Bengio / Aaron Courville
- 官方页面：<https://mitpress.mit.edu/9780262035613/deep-learning/>
- 中文版：<https://deeplearning.ac.cn/>
- 简介：深度学习理论经典教材。
- 推荐理由：如果想认真理解优化、表示学习、正则化、序列模型等理论，这本书非常重要。
- 适合人群：进阶 / 做科研的同学。
